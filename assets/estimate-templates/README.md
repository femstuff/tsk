# Шаблоны сметы (строительно-отделочные работы)

Эталонные файлы с [assistentus.ru](https://assistentus.ru):

| Файл | Назначение |
|------|------------|
| `blank-smety-na-stroitelno-otdelochnye-raboty.doc` | Пустой бланк для заполнения |
| `obrazec-smety-na-stroitelno-otdelochnye-raboty.doc` | Заполненный образец |

Формат: **Word 97–2003 (.doc)**. Для программной подстановки полей позже лучше конвертировать бланк в **.docx** с именованными закладками или плейсхолдерами `{{projectName}}` и т.д.

В рабочую систему шаблон загружается через админку (**Документы → шаблоны**) или `POST /api/v1/document-templates` с `category=estimate`.

**Продакшен:** при старте backend автоматически создаёт шаблон **Word `.docx`** (если ещё нет docx-сметы) с плейсхолдерами `{{projectName}}` и т.д. Итоговый файл после подтверждения — тоже **`.docx`** (открывается в Word).

Файлы `.doc` (Word 97–2003) в этой папке — эталон с assistentus.ru; для автозаполнения используется сгенерированный **docx**-шаблон в коде (структура формы № 4). Конвертацию вашего бланка `.doc` → `.docx` с теми же полями можно сделать в Word вручную и загрузить через админку.
