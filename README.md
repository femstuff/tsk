# Транскрипция голосовых сообщений

Проект состоит из двух сервисов:
1. **Python API** (Whisper) - транскрибирует аудио
2. **Go Telegram бот** - принимает голосовые и отправляет на транскрипцию

## Запуск через Docker

```bash
# Собрать и запустить
make build
make up

# Остановить
make down

# Посмотреть логи
make logs
